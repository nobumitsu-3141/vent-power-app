# 人工呼吸 & Mechanical Power シミュレーター

GOTO ANESTHESIA TOOLKIT · VENTILATION — 麻酔の換気を学び/評価する単一ページ Web アプリ（PWA・オフライン対応）。

**公開URL（iPhone/iPad は Safari で開き「ホーム画面に追加」）:**
https://nobumitsu-3141.github.io/vent-power-app/

## タブ
- **患者評価（実測）**: 麻酔器の表示値から R・C・τ・駆動圧・auto-PEEP・推定MP・PBW/(VT/kg) を推定（吸気フロー2点法、I:E・ポーズ対応）
- **換気・MP**: VCV/PCV シミュレーター（MP内訳・PEEP除外・Stress Index・auto-PEEP・圧カーブ注釈・ETT抵抗早見・PBW→VT）
- **BEST PEEP**: Recruitment-to-Inflation (R/I) ratio ＋ decremental PEEP trial
- **換気量・CO₂**: 換気量・死腔（体格×デバイス×回路部品）・目標EtCO₂逆算・pH推定
